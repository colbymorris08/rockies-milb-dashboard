HOW THIS SHAPED THE MODEL

Release extension — Mean release extension across this board’s Coors pitches (2021–24). The XGBoost uses log(extension); the Coors-adjusted path scales it by α×1.18.
Four-seam VAA & arm slot — VAA is computed at plate crossing from Statcast trajectory (vx0/vy0/vz0 + ax/ay/az); four-seam averages should sit in a realistic negative range. Arm angle is Statcast’s release metric (0° sidearm → 90° over-the-top).
Pitch mix & slider family — “Primary pitch” is each pitcher’s most-thrown pitch type in the scored Coors sample; the line shows how many of this board’s arms lead with FF, SL, SI, etc. Slider + sweeper share is SL+ST pitches ÷ all scored pitches (gyro/bullet shape work shows up in this family).
Stuff+ snapshot — Mean sea-level and Coors-adjusted index scores and mean Δ for this leaderboard; the table strip above leans on mechanics and mix.
Qualification guardrails 40 IP all · 80 IP Rockies 10-year Statcast Coors workload (2015–24) gates the composite; the Stuff+ scores themselves use 2021–24 Coors pitches.
COORS STUFF MODEL (INTERNAL XGBOOST)

Two gradient-boosted models predict pitch quality from Statcast: one on all parks (“sea-level” features) and one on the same pitch rows with Coors-specific scaling (α) on movement, extension, and release context. Outputs are standardized to a 100/10 “Stuff-style” scale, then paired with long-run Coors performance for the composite leaderboard. The feature set is informed by common public research threads—velocity and movement decomposition, release-side context, count and handedness, spin axis—but it is not a direct port or credit to any single external Stuff product; weights and stacking are mine.
